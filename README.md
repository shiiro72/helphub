# HelpHub

## Quick start

Install dependencies:

```bash
npm install
```

Run the Next.js app:

```bash
npm run dev
```

Open the app at [http://localhost:3000](http://localhost:3000).

Run Storybook:

```bash
npm run storybook
```

## Project structure

- `src/components` — reusable components
- `src/stories` — Storybook stories for components
